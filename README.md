# image-processing-c
C-based image processing application for RGB channel separation and BMP file handling

### Features
- Reads and validates 24-bit BMP files
- Extracts Red, Green, and Blue channels
- Generates separate 8-bit grayscale BMP images
- Applies channel-specific color palettes
- Handles binary file I/O and memory management

### Technologies
- C
- Binary file handling
- BMP file format

### Output
- red8.bmp
- green8.bmp
- blue8.bmp
